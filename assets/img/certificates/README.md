# Certificates Folder

Upload your certificate files here and they will appear automatically in the Certificates page.

Supported formats:
- `.png`
- `.jpg`
- `.jpeg`
- `.webp`
- `.gif`
- `.svg`
- `.pdf`

Tips:
- Use clean filenames (example: `cap-certified-appsec-practitioner.png`)
- Landscape images with similar aspect ratios look best
- For badges, you can also upload files to `/assets/img/badges/`
