# Assets Directory

## Resume/CV

Please place your resume PDF file here as `resume.pdf`.

**File name:** `resume.pdf`
**Path:** `/assets/resume.pdf`

The resume button on the website points to `/assets/resume.pdf`.

## Instructions

1. Create or export your resume as a PDF file
2. Name it exactly: `resume.pdf`
3. Place it in this `/assets/` directory
4. The download button will automatically work

---

**Note:** Make sure the PDF is professional and up-to-date with your latest skills, projects, and experience.
